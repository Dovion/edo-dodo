## 2️⃣ Как запустить приложение

### 📥 Шаг 1. Получите файлы проекта
Вы получите проект одним из способов:
- **Вариант А:** Вам передадут папку с проектом (`edo-dodo`) — просто распакуйте её в удобное место.
- **Вариант Б:** Вам предоставят ссылку на репозиторий — перейдите по ней и скачайте архив (кнопка **Code** → **Download ZIP**), затем распакуйте.

### ▶️ Шаг 2. Запустите одним кликом

#### Для Windows:
1. Откройте папку проекта
2. Нажмите правой кнопкой мыши в пустом месте → **«Открыть в Терминале»** (или «Open PowerShell window here»)
3. Введите команду и нажмите **Enter**:

   ```bash
   docker compose up -d
#### Для macOS:
1. Откройте приложение Terminal
2. Перейдите в папку проекта
   ```bash
   cd ~/Downloads/edo-dodo
3. Введите команду и нажмите **Enter**:

   ```bash
   docker compose up -d
